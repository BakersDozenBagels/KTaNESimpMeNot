On the Subject of Simp Me Not

No, this is not a simpleton.

Note: This module does not have an exclamation point. If it does, you are looking at the wrong manual.

Once every other non-ignored module has been solved, this module will be ready for input. Attempting to input before this time or once you have already successfully input will result in a strike. To input, press the button the correct number of timeswith little delay between presses. After too much time has passed from your last press, the input will be submitted. The correct number of presses is the number of non-ignored modules on the bomb. An incorrect submission will result in a strike.
